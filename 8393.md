# 문제 제목: 1부터 n까지 합 구하기

## 문제 정보
- **출처:** https://www.acmicpc.net/problem/8393
- **난이도:** Nomal

## 문제 설명
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오

## 해결 과정
구글링
### 접근 방법
input 을 이용해 n을 할당 받고
for문을 이용하여 1부터 n번까지 반복을한뒤
sum 에 i 를 더하는 할당을 해서
print를 이용해 sum 결과값을 출력하였습니다.

## 코드
```python
n = int(input())

sum = 0

for i in range(1, n+1):
    sum += i
    
print(sum)
